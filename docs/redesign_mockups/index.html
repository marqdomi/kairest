Propuesta de Rediseño — CasaLeones POS v7

Una UI que se siente
premium y moderna

Mockups de rediseño para los dashboards principales de CasaLeones POS. Cada pantalla fue diseñada con un lenguaje visual contemporáneo, jerárquico y orientado a la eficiencia operativa.

Diagnóstico

¿Por qué rediseñar?

La UI actual es funcional pero acumuló deuda visual en 6 sprints. Estos son los principales problemas que resuelve el nuevo diseño.

🎨

Paleta inconsistente

Mezcla de colores Bootstrap, variables custom y hardcoded. El nuevo sistema usa un solo archivo de tokens como fuente de verdad.

📊

KPIs sin jerarquía visual

Los cards KPI actuales tienen pesos iguales y no guían la mirada. El nuevo diseño usa contraste, tamaño y color para priorizar lo urgente.

🌙

Dark mode inconsistente

El modo oscuro actual tiene ~291 líneas de overrides. El nuevo diseño lo hereda automáticamente de los tokens CSS con ~50 líneas.

⚡

Navegación confusa

El sidebar no tiene secciones claras, mezcla acciones operativas y administrativas. El rediseño organiza por flujo de trabajo real.

📋

Tablas desnudas

Sin búsqueda, filtros ni paginación en el 80% de las vistas CRUD. El sistema nuevo incluye estos como parte del componente base.

🦁

Identidad de marca diluida

El rojo CasaLeones (#C41E3A) aparece de forma tímida. El nuevo diseño lo usa con confianza como ancla cromática en todo el sistema.

Sistema de Color

Nueva paleta propuesta

Dos colores primarios de marca (Rojo León + Dorado) sobre grises neutrales de alta fidelidad y cuatro semánticos para estados.
